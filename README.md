# Electric-Vehicle-Recharge-Bunk
A web application to manage EV recharge bunk locations, slots, and users.

 Technologies Used:
- HTML
- CSS
- JavaScript
- Firebase (Auth + Database)

 Modules :
 Admin
- Login / Register
- Create EV Bunk Location
- Manage Bunk Details
- Manage Recharge Slots

 User :
- Login / Register
- Search Nearby EV Bunk
- View Slot Availability

Project Workflow:
1. User registers and logs in
2. User searches nearby EV bunk
3. Admin manages locations and slots

How to Run Project :
1. Download project folder
2. Open Firebase console and create project
3. Enable Authentication (Email/Password)
4. Enable Firestore Database
5. Add Firebase config in `firebase.js`
6. Run using Live Server in VS Code
